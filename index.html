Project README

Logistics Management Dashboard

This project is a role-based logistics and fleet management dashboard created as a school project.

Roles Included

  • Owners
  • HR/Admin
  • Employees
  • Drivers
  • Mechanics
  • Dispatch
  • Safety Compliance
  • Support

Main Features

  • Role-based dashboards
  • Shared data across modules
  • Owner analytics and smart AI insights
  • Dispatch and driver load tracking
  • HR complaint and investigation workflow
  • Mechanics work order tracking
  • Safety compliance records
  • Support ticket management
  • AI Logistics Assistant
  • Role-aware actions and report generation

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Chart.js
  • GitHub Pages

Purpose

The purpose of this project is to demonstrate how a logistics company can manage fleet operations through a role-based dashboard system where each department accesses relevant data and workflows.
